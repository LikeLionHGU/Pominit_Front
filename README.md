## SeaZle : 포항 해양 스포츠의 모든 것


✨ 기능



### 👀 다양한 강습소 정보 한눈에

카드 리스트로 강습소 핵심 정보(이름, 위치, 가격대, 평점)를 빠르게 스캔

필터/검색으로 스포츠 종목·지역·가격·평점 기준 정렬

상세 페이지로 이동해 더 많은 정보를 확인
